# BM 531 — Biomedical Instrumentation Study Site

A pretty, pastel-themed visual study guide for BM 531.

## Files
- `index.html` — Main page
- `style.css` — All styles
- `script.js` — Scroll animations

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `bm531-notes`)
2. Upload all 3 files to the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save**

Your site will be live at:
`https://YOUR-USERNAME.github.io/bm531-notes/`

That's it! No build tools needed.
